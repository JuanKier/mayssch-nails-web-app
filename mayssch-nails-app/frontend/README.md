# Mayssch Nails - Gestión de Citas

Aplicación web para la gestión de citas de uñas de Mayssch Nails.

## Características

- 📅 Calendario de citas con horarios predefinidos
- 👥 Gestión de clientas
- 💅 Gestión de procedimientos
- 📊 Reportes y estadísticas
- 💰 Contabilidad con gráficos financieros
- 📦 Control de inventario
- 📱 Diseño responsive (móvil y tablet)

## Tecnologías

- React 19
- Vite
- Tailwind CSS
- Flask (Backend)
- SQLite (Base de datos)

## Instalación

```bash
# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev
```

## Despliegue

Esta aplicación está diseñada para funcionar como PWA (Progressive Web App) y puede instalarse en dispositivos móviles.
